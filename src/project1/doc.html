Goals:

The goal for my site as of right now is to display projects both eduactional and personal in a format similar to a portfolio. The audience for this site will be anyone who wants to know more about me or my works.

Notes:

Changes from Phase 1 → Phase 2

  • Changed the color scheme of website to be more inline with personal tastes and less saturated with color.
  • Added a nav bar with page links to assist navigation on mobile devices and desktop when more content is added that lengthens the site.
  • Reduced overall usage of id elements for items and utilized more class based selectors
  • Added an about me grid area which is moved depending on site size.
  • Implemented a mobile first design where the site utilizes media queries to override style elements for desktop

Challenging parts:

Phase 1

  • Positioning the floating boxes of content
  • Properly setting the depth of content boxes so that they create a 3D effect.
  • Using a grid to alight elements but integrate responsivity into elements

Phase 2

  • Finding proper sizing for headshot image so that it does not get blown up when on a desktop screen. To fix this I chose to utilze a two column grid and only place the image on the left with an about me/bio area allocated on the right.
  • Properly formatting the content section where projects will be displayed. I wanted objects to be more floaty and less formatted. Actually acomplishing what I wanted was pretty difficult and it is still not quite there.
  • Deciding on a proper format for the website. I had a hard time deciding how much I wanted to change the content from phase 1 → phase 2.
