Goals:

For my project I want to create a webapp which utilizes the Jikan Unofficial MyAnimeList API to present the user with recommendations based on an anime they previously watched as a core functionality. The user would only need to type in an anime, and it will present them with similar options. Alongside this, there would be other options to get completely random anime/manga and/or present the user with news regarding the anime provided. The target audience will be people who have watched anime or read manga before and are looking for something else similar to what they search. Also, since the site will have the functionality to provide random anime/manga potentially based off of other filters like genre or rating, another portion of the audience could be users who are watching or reading for their first time. Depending on how the core functionality and other search options go, I could even implement features found in MAL itself like a list for things you've watched(of course only if I have time)
